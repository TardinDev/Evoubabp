# Screenshots TikTok Clone

Ce dossier doit contenir les captures d'écran de l'application TikTok Clone.

## Images requises :

### 1. feed.png
- **Titre**: Feed de vidéos
- **Description**: Défilement vertical infini avec vidéos en autoplay
- **Contenu suggéré**: Capture d'écran montrant le feed principal avec une vidéo en lecture, les boutons d'action (like, comment, share) sur le côté droit, et le nom d'utilisateur en bas.

### 2. camera.png
- **Titre**: Caméra d'enregistrement
- **Description**: Interface de capture vidéo avec effets et filtres
- **Contenu suggéré**: Capture d'écran de l'interface de caméra avec le bouton d'enregistrement rouge en bas, le bouton pour retourner la caméra en haut à droite.

### 3. profile.png
- **Titre**: Profil utilisateur
- **Description**: Page profil avec grille de vidéos et statistiques
- **Contenu suggéré**: Capture d'écran du profil utilisateur avec photo de profil, statistiques (abonnements, abonnés, j'aime), bouton s'abonner, et grille de vidéos en dessous.

### 4. comments.png
- **Titre**: Système de commentaires
- **Description**: Bottom sheet avec commentaires et réponses
- **Contenu suggéré**: Capture d'écran du bottom sheet de commentaires avec la liste des commentaires, avatars, boutons like et répondre.

## Format recommandé :
- Format: PNG ou JPG
- Ratio d'aspect: 9:16 (format mobile vertical)
- Résolution recommandée: 1080x1920 pixels
- Taille de fichier: < 500 KB par image

## Notes :
- Les images doivent être claires et de haute qualité
- Assurez-vous que le texte est lisible
- Évitez les informations personnelles sensibles dans les captures d'écran
- Vous pouvez utiliser des données de démonstration

## Fallback :
Si une image n'est pas trouvée, un placeholder avec le titre de la section sera affiché automatiquement.
