# Instructions pour ajouter l'image TikTok.png

## Emplacement
`/public/imagesFormations/TikTok.png`

## Spécifications
- **Type**: Image composite montrant plusieurs écrans de l'application
- **Format**: PNG
- **Dimensions recommandées**: 1920x1080 pixels (paysage)
- **Taille de fichier**: < 2 MB

## Contenu suggéré
L'image devrait montrer une maquette avec 3-4 écrans côte à côte :
1. **Écran du feed** - avec une vidéo en lecture
2. **Écran de profil** - avec statistiques et grille de vidéos
3. **Écran de caméra** - interface d'enregistrement
4. **Écran de commentaires** (optionnel)

## Alternatives
Si vous n'avez pas encore l'image, vous pouvez :
1. Créer une maquette avec Figma ou Canva
2. Prendre des captures d'écran de votre application et les combiner
3. Utiliser un outil comme Shotsnapp ou MockUPhone pour créer des mockups professionnels

## Note
Le projet TikTok utilisera la galerie de screenshots (dans `/public/imagesFormations/tiktok/`) pour afficher les détails, donc cette image principale est optionnelle mais recommandée pour l'aperçu général.
